# Prime Tables Interview Sample

## Dependencies

    NodeJS >= 4.5.0
    NPM in your PATH
    Google Chrome Browser

## Quick start

    $ npm install -g http-server
    $ npm install -g bower
    $ npm install -g karma-cli
    $ npm install
    $ bower install

## Running tests

    $ karma start ./spec/karma.conf.js

## Running the site

    $ http-server ./app/
